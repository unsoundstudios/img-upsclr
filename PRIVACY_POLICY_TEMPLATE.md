## Scope

This policy describes how IMG-UPSCLR handles data for desktop and web deployments.

## Desktop Application

- Image files are processed locally on the user’s machine.
- No telemetry is required by default in this codebase.
- No user account data is collected by default.

## Website Deployment

Operators of the web version may process uploaded images and related metadata (file names, timestamps, job status) to provide the service.

Recommended operator controls:

- automatic cleanup of temporary job files (`MAX_JOB_AGE_HOURS`)
- TLS for data in transit
- strict access controls for server storage
